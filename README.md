## 📖 Overview

The **Apartment Building Management System** is a cross-platform mobile application that streamlines the management of apartment buildings. It provides tools for administrators, residents, and staff to handle daily operations efficiently.

### Key Features

- 🏠 **Apartment Management** — Track units, floors, and building info
- 👤 **Resident Management** — Resident profiles, move-in/move-out tracking
- 💰 **Fee & Payment Management** — Service fees, utility bills, payment history
- 📋 **Service Requests** — Maintenance requests, complaint handling
- 🔔 **Notifications** — Announcements and alerts for residents
- 📊 **Reports & Statistics** — Dashboard with key metrics

## 📋 Prerequisites

Before you begin, ensure you have the following installed:

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (≥ 3.11.5)
- [Dart SDK](https://dart.dev/get-dart) (included with Flutter)
- [Android Studio](https://developer.android.com/studio) or [VS Code](https://code.visualstudio.com/) with Flutter extension
- An Android emulator or physical device

Verify your setup:

```bash
flutter doctor
```

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/duongk18FPTU/apartment-management-prm393.git
cd apartment-management-prm393
```

### 2. Install dependencies

```bash
flutter pub get
```

### 3. Run the application

```bash
# Run on connected device / emulator
flutter run

# Run on a specific device
flutter run -d <device_id>
```
